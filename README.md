
# AI Career Recommendation & Skill Gap Analyzer

An intelligent front-end web application that analyzes user skill levels and recommends the most suitable career path using a weighted scoring algorithm and skill gap analysis.

This project demonstrates explainable AI logic, decision systems, and frontend engineering for interview showcase purposes.

---

## Live Demo

Open the HTML file in any browser:

```
Career Recommendation & Skill Gap Analyzer.html
```

No backend required. Fully frontend powered.

---

## Features

### Career Recommendation Engine

Analyzes user skills:

* Programming
* Database / SQL
* Problem Solving
* Computer Networks

Uses weighted scoring algorithm to recommend:

* Software Engineer
* Data Analyst
* Cybersecurity Analyst

---

### Skill Gap Analyzer

Identifies weak areas where skill level < 6

Displays:

* Missing skills
* Improvement areas

---

### Confidence Score Visualization

Shows:

* Career match percentage
* Visual progress bar

---

### Personalized Learning Roadmap

Recommends technologies to learn for selected career

Example:

Software Engineer roadmap:

* DSA
* Java / Python
* Spring / React

---

## AI Decision Logic

The system uses weighted scoring model:

Example:

Software Engineer:

```
Score =
Programming × 0.45
+ Database × 0.20
+ Logic × 0.35
```

Final score converted into percentage.

Highest score = Best career recommendation
